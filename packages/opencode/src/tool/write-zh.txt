将文件写入本地文件系统。

用法：
- 如果目标路径已有文件，此工具将覆盖该文件。
- 如果是已有文件，你必须先使用 Read 工具读取文件内容。如果未先读取文件，此工具将失败。
- 始终优先编辑代码库中的已有文件。除非明确要求，否则绝不创建新文件。
- 除非用户明确要求，否则绝不主动创建文档文件（*.md）或 README 文件。只有在用户明确要求时才创建文档文件。
- 除非用户明确要求，否则仅使用表情符号。除非被要求，否则避免在文件中添加表情符号。
- 对于大文件，建议分多个小步骤写入内容，而不是一次性写入：单次过大的写入可能超出输出限制而被截断，导致文件不完整或损坏。先使用此工具写入初始部分，然后通过后续的 Edit 调用添加剩余部分。
